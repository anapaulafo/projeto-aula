Projeto: Aplicativo para acompanhar progresso do usuário ao assistir os filmes indicados ao Oscar 2025.

Orientações:

Para rodar este projeto:

1 - abra o terminal do VSCode, e instale as dependências do express;

> npm install express

2 - entre na pasta backend e execute o servidor na porta 4000;

> cd backend
> node server.js

3 - em um novo terminal, instale as dependências do front;

> npm install

4 - execute o projeto react.

> npm run dev
